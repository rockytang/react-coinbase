# Prices Dashboard
A React/Redux app using the Coinbase API to refresh BTC and ETH prices every 60 seconds.

**This dashboard is deployed on** https://rockytang.github.io/react-coinbase/  
  
## Instructions

#### Install
```bash
npm install
```

#### Run the app
```bash
npm start
```

#### Use the app
http://localhost:3000/
